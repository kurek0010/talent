# Karty walutowe Talenta

Rozwinięcie audytu `../TALENTY_swiatowe_v0.1.md`: jedna karta na walutę, w formacie aneksu do `../POLITYKA_danych_v0.1.md` — konkretne serie, kalendarz publikacji, ryzyka specyficzne dla kraju i werdykt. Wpisy ✻ = z wiedzy, do potwierdzenia u źródła przed wdrożeniem danego kraju.

| Karta | Poziom | Werdykt jednym zdaniem |
|---|---|---|
| [TALENT-PLN](TALENT-PLN.md) | A | rdzeń projektu; mediana z rejestrów ZUS+MF jako przewaga |
| [TALENT-USD](TALENT-USD.md) | A | najlepsze dane świata; wszystko w FRED, wdrożenie natychmiastowe |
| [TALENT-GBP](TALENT-GBP.md) | A | bardzo dobra; administracyjne dane PAYE jako drugie źródło płac |
| [TALENT-JPY](TALENT-JPY.md) | A− | pełne dane miesięczne; lekcja skandalu 2018 wpisana w tripwire |
| [TALENT-EUR](TALENT-EUR.md) | B+ | najlepsza noga cen świata (20 urzędów), noga płac tylko kwartalna |
| [TALENT-CHF](TALENT-CHF.md) | B | noga płac roczna — najsłabsza w gronie; kraj o najstabilniejszych płacach łagodzi problem |
| [TALENT-CNY](TALENT-CNY.md) | C | **duża aproksymacja** — publikowalny wyłącznie z szerokim pasmem błędu, nierekomendowany do rozliczeń krzyżowych |
